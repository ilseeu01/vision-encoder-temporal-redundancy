# This project uses FOUR separate conda environments because the vision models
# require incompatible torch / transformers versions. There is no single set of
# deps that works for all 9 models — install the per-env file into each env:
#
#   requirements-qwen3vl.txt   CLIP, DINOv2, Qwen2.5-VL, Qwen3-VL, LLaVA-OV   (py3.11, transformers 4.57)
#   requirements-internvl.txt  InternVL2.5 / InternVL3                        (py3.9,  transformers 4.37.2)
#   requirements-sam2.txt      SAM2.1  (+ SAM2 source install)                (py3.10)
#   requirements-sam3.txt      SAM3    (+ SAM3 source install)                (py3.12)
#
# Versions below are the exact ones used (pip freeze of the live envs). See README "Setup".
#
# Pure-analysis / plotting (common.py, plots, aggregation) only needs:
#   numpy, scipy, matplotlib, pillow, av (or decord)
